页面模板 ​

Keep 主题对友链（Links）、相册（Photos）这两个页面模板做了内置的样式美化，只需按如下教程，便可得到漂亮的页面。

友链 Links ​

效果图 ​

配置流程 ​

  1. 创建友链（Links）页面，在 Hexo 项目根目录下使用如下命令即可自动创建。

    sh
    cd your-hexo
hexo new page links
    1
    2

    注意

    请勿删除或修改 source/links/index.md 文件的 title 属性

    ---
title: links
date: 2020-05-13 14:16:07
---
    1
    2
    3
    4

  2. 在 Keep 主题配置文件 _config.yml 的 menu 配置项里面开启 Links 导航菜单。

    yaml
    menu:
  Home: /
  Links: /links
  # ......
    1
    2
    3
    4

  3. 在你的 Hexo 项目的 source 文件夹里增加 _data 文件夹。

    提示

    如果已经有了 _data 文件夹，则忽略这一步。

  4. 在 _source/_data 文件夹里新建 links.yml 文件。

  5. 在 links.yml 文件里按如下格式增加你的友链数据。

    yaml
    - title: 朋友们 1

- name: XPoet
  link: https://xpoet.cn/
  description: 懒惰是程序员第一生产力
  avatar: https://cdn.statically.io/gh/XPoet/image-hosting@master/common-use/avatar.jpg

- name: 不知名艺术家
  link: https://jinzhanqi.com/
  description: love artist, love code.
  avatar: https://s3.ax1x.com/2020/11/17/DVvkB4.jpg

- ......  
- ......  
- ......

相册 Photos ​

效果图 ​

配置流程 ​

  1. 创建相册（Photos）页面，在 Hexo 项目根目录下使用如下命令即可自动创建。

    sh
    cd your-hexo
hexo new page photos
    1
    2

    注意

    请勿删除或修改 source/photos/index.md 文件的 title 属性

    ---
title: photos
date: 2023-11-08 14:16:07
---
    1
    2
    3
    4

  2. 在 Keep 主题配置文件 _config.yml 的 menu 配置项里面开启 Photos 导航菜单。

    yaml
    menu:
  Home: /
  Photos: /photos
  # ......
    1
    2
    3
    4

  3. 在你的 Hexo 项目的 source 文件夹里增加 _data 文件夹。

  4. 在 source/_data 文件夹里新建 photos.yml 文件。

  5. 在 photos.yml 文件里按如下格式增加图片数据。

    yaml
    - url: https://xpoet.cn/images/logo.png
  name: XPoet

- url: /images/photos/img-1.png
  name: img1

- url: /images/photos/img-2.png
  name: img2

- ......  
- ......
