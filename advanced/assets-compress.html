资源压缩 ​

Keep 主题的源码中，HTML、CSS、JavaScript、图片等资源文件都是没有混淆压缩的。绝大部分的 Hexo 主题也是如此。庆幸的是，可以找到相关的 Hexo 插件来解决这一问题。

hexo-all-minifier 集成了 html-minifier、clean-css、uglify 和 imagemin 等插件，可以对 Hexo 主题资源进行有效压缩，只需安装和简单配置即可使用。

  1. 在 Hexo 项目根目录下安装 hexo-all-minifier

    sh
    cd your-hexo
npm install hexo-all-minifier
    1
    2

  2. 在 Hexo 配置文件 _config.yml 文件中，添加以下配置项

    yaml
    all_minifier: true
    1

你在使用 hexo-all-minifier 时遇到问题或者想进行更加详细的配置，请查看插件的 官方文档
