Mermaid 流程图 v3.8.3 ​

如果要在文章中绘制 Mermaid 流程图，可以使用 Hexo 插件 hexo-filter-mermaid-diagrams。

Mermaid 配置流程 ​

  1. 在 Hexo 项目根目录下安装插件 hexo-filter-mermaid-diagrams

    bash
    cd your-hexo
npm install hexo-filter-mermaid-diagrams
    1
    2

  2. 在 Hexo 配置文件 _config.yml 增加如下配置

    yaml
    mermaid:
 enable: true
 version: 10.5.0
    1
    2
    3
    • enable 控制是否启用 Mermaid
    • version 选择 Mermaid.js 版本

使用示例 ​

完成 Mermaid 插件配置之后，即可在文章中使用 Mermaid 代码块进行绘制流程图。

示例一 ​

markdown
```mermaid
graph TD;
    A-->B;
    A-->C;
    B-->D;
    C-->D;
```
