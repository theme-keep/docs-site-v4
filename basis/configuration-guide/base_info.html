配置指南的目录结构跟主题配置文件 _config.yml 的配置项保持一一对应，方便查看。

base_info ​

base_info 配置项用于设置你的博客网站基本信息。

yaml
base_info:
  primary_color: "#0066cc"
  title: Keep Theme
  author: Keep Team
  avatar: /images/avatar.svg
  logo: /images/logo.svg
  favicon: /images/logo.svg

primary_color ​

primary_color 用于设置博客网站的主题颜色，支持 十六进制、rgb 和 rgba 颜色格式。
例如：#0066cc、rgb(0, 102, 204) 和 rgba(0, 102, 204, 0.8)。

获取主题配色推荐网站：

  • Flat UI Colors
  • Web 安全色
  • Adobe Color CC

示例：

  • 蓝色主题 #0066cc

    yaml
    primary_color: "#0066cc"
    1

  • 红色主题 #c0392b

    yaml
    primary_color: "#c0392b"
    1

avatar ​

avatar 用于设置博客网站的作者头像，将显示在文章内容页，如下图。

不填则不显示作者头像

avatar 头像设置示例：

  • 使用本地图片路径

    yaml
    # your-hexo/source/images/avatar.svg

avatar: /images/avatar.svg
    1
    2
    3

  • 使用图片链接

    yaml
    avatar: https://keep.xpoet.cn/images/avatar.svg

关于图片的设置：

你可以使用 本地图片路径（本地图片必须是存储在 your-hexo/source/images/ 路径下的图片）或 图片链接 。

  • 本地图片路径

    yaml
    # your-hexo/source/images/avatar.svg

avatar: /images/avatar.svg

  • 图片链接

mode v4.1.3 ​

mode 是 v4.1.3 新增的隐藏配置项，用于指定博客网站的主题模式，使其不跟随系统的主题模式而改变。可选 light（白昼模式）和 dark（暗夜模式）。

使用示例：

yaml
# 将博客网站的主题模式设置为暗夜模式
base_info:
  mode: dark  # Option values: light | dark
1

提示

除了设置 mode，你也可以随时通过点击右下角工具栏的切换按钮手动切换主题模式。
