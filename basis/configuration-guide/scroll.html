Table of Contents for current page

scroll ​

scroll 设置博客网站页面在滚动时的样式。

progress_bar ​

progress_bar 页面滚动时在网站顶部显示进度条，可选值：true 或 false。

percent ​

percent 页面滚动时在右下角工具栏显示进度百分比，可选值：true 或 false。

小提示

鼠标移到百分比按钮处会显示向上箭头图标，点击即可回到顶部。

hide_header v4.0.0 ​

hide_header 在页面向下持续滚动时，设置是否自动隐藏 Header 头部，可选值：true 或 false。 该配置项的值默认为 true。

示例：

  • hide_header: true 隐藏 Header

  • hide_header: false 显示 Header
