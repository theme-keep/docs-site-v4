启动模板 ​

hexo-theme-keep-starter 是由 Keep 主题官方提供的快速启动模板，我们可以直接拿来快速创建一个内置 Keep 主题的 Hexo 博客项目，开箱即用，然后在此基础上配置出属于自己的独一无二的 Keep 主题博客，非常适合刚接触 Hexo 和 Keep 主题的小伙伴。

下载启动模板 ​

你可以通过以下两种方式来下载 Keep 启动模板：

使用 Git 命令克隆 ​

sh
git clone https://github.com/theme-keep/hexo-theme-keep-starter
1

直接下载 ​

点击这里下载 >> Keep 主题快速启动模板

使用启动模板 ​

安装依赖 ​

Keep 主题快速启动模板下载完成之后，需要安装项目依赖才能成功运行起来。

sh
cd hexo-theme-keep-starter
npm install
1
2

运行项目 ​

项目依赖安装成功之后，使用如下命令即可运行项目，随即便可在浏览器上看到包含 Keep 主题的 Hexo 博客网站。

sh
cd hexo-theme-keep-starter
npm run server
1
2

Keep 主题博客网站效果图：

注意

使用 Keep 主题必须确保你的 Node.js 版本 >= 14.0，当你的 Hexo 项目运行时报错或发出警告时，请注意升级你的 Node.js 版本。

配置主题 ​

通过修改启动模板中的 Hexo 配置文件 _config.yml 和 主题配置文件 keep.yml，来实现自己的博客个性设置。

配置文件路径 ​

  • Hexo 配置文件 hexo-theme-keep-starter/_config.yml

  • Keep 主题配置文件 hexo-theme-keep-starter/source/_data/keep.yml

注意事项 ​

  • 为确保你的 Keep 主题配置项完全生效，请在 source/_data/keep.yml 路径下填写主题的相关配置。

  • 请勿随意移动 source/_data/keep.yml 文件的路径，确保 keep.yml 在 source/_data/ 路径下。

  • 请勿随意更改 source/_data/keep.yml 文件中的配置项名称，填写配置项的值之后，确保 YAML 文件的缩进是否正确。
